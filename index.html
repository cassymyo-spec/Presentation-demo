— Core Capabilities
Live Sensor Streaming
Real-time temperature, flow rate, and pump status data streamed directly from field sensors every 5 seconds.
Remote Pump Control
Start or pause borehole pumps remotely from the dashboard — with immediate state confirmation from the field unit.
Intelligent Alerting
Threshold-based alerts for critical temperature, abnormal pump cycles, and prolonged offline states — with severity levels.
Historical Analytics
View temperature trends and pump uptime over 24h, 7d, or 30d windows with interactive time-series charts.
Role-Based Access
Separate operator and admin roles ensure field technicians have the right permissions without exposing system settings.
Any-Device Access
Fully responsive web interface works on desktop, tablet, and mobile — accessible from any site or office location.
